name: Lynx intro title-screen menu styling
description: How the intro main-menu items should look (user aesthetic preference)
---

- The intro title screen's menu items must be **text-only** — no bordered/outlined "button" boxes.
- **Why:** the user explicitly disliked the "ugly neon green outline" on boxed buttons (2026-07) and wants the HOUND game-menu look: a bottom-left vertical list of stylised text.
- **How to apply:** keep menu labels as plain text in the gothic display face (`font-gothic` / Grenze Gotisch), positioned bottom-left. For hover/focus/active use a subtle lime *text color* + a small `►` marker + a soft feathered radial glow behind the item — never a solid box or a visible rectangular border. Lime (brand) stays the accent, not gold.
